Что мы сделали?

  • Принцип визуализации: использовали VirtualBox для создания изолированных виртуальных машин.
  • Работа с ОС: Kali Linux, Windows, Alt Linux, Astra Linux.
  • Работа с командной строкой: освоили базовые команды.
  • Настройка межсетевого экрана: программный комплекс, который отслеживает сетевые пакеты, блокирует или разрешает их прохождение.
  • Настройка сети:
    • NAT: доступ к интернету
    • Сетевой мост: в одной сети с хостом
    • Внутренняя сеть: изолированное тестирование
  • Wireshark: анализ сетевого трафика.
  • nmap: сканирование сети.
  • Использовали: ScanOval, FlanSCAM, OpenVAS.
  • Backdoor: вредоносная программа для получения доступа к системе с обходом аутентификации.
  • ARP-Spoofing: протокол, позволяющий определить MAC-адрес по известному IP-адресу.

С помощью Cyber Backup создали первую резервную копию системы.
